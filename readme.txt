WordPress theme SANGO dedicated plugin. You can easily change the settings.

== Description ==


SANGO＋(プラス)は、WordPressテーマ「SANGO」専用のプラグインです。
プラグインのインストールが完了すると、WordPress管理画面の「外観>カスタマイズ」パネルに「SANGO +設定」が表示され、そのメニューからチェックボックスなどで設定できます。

使える機能は以下の通りです。
* 追従する目次のボタンの表示
* 固定ページでタグを利用可能に
* 追従する目次ボタンのカラーピッカー追加
* アイキャッチ画像を全幅化
* リンクカードをフェードイン
* 指定した画像でローディングアニメーション
* 記事一覧のカードにタグを表示
* 7種類のGoogleフォントを追加
* コピープロテクト機能


== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
2. In the browser input box, type “My Custom Style Css Manager”.
3. Select the “My Custom Style Css Manager” plugin and click “Install”.
4. Activate the plugin.

OR…

1. Download the plugin from this page.
2. Save the .zip file to a location on your computer.
3. Open the WP admin panel, and click “Plugins” -> “Add new”.
4. Click “upload”.. then browse to the .zip file downloaded from this page.
5. Click “Install”.. and then “Activate plugin”.
